# Implementation-and-experimental-evaluation-of-inverse-kinematics
# Helga - Reach Alpha 5 Thesis Data

This repository contains supplementary data for the master's thesis "Implementation and experimental evaluation of inverse kinematics for robotic arm pick-and-place on an unmanned surface vehicle".

## Files:
- `CSV files.zip` – Raw measurement data from IK accuracy tests (CSV files). Each file contains joint angles and end-effector errors for five IK methods across eight test points.
- `Plot Figures.zip` - Figures of different plotted data
- `Python code for ploting.zip` - Python code that was used for ploting and processing the raw data from the CSV files
- `Pick-and-place demo.mp4` – Video recording of the complete pick-and-place sequence
